## 🛠️ Tech Stack & Key Attributes
*   **Frontend Architecture:** Vanilla HTML5, CSS3 (using modern `color-mix()` syntax and CSS variables), Vanilla JavaScript (ES6+).
*   **Asset Footprint:** Zero dependencies, zero compilation steps, lightning-fast rendering times.
*   **Unique Features:** Native system-driven light/dark mode toggling, interactive viewport tracking via `IntersectionObserver`, and form submission pipeline powered by Formspree API.
